# **Events**

The Bet module emits the following events

## *MsgPlace*

|  Type         |  Attribute Key|    Attribute Value    |
|:-------------:|:-------------:|:---------------------:|
| bet_place     | bet_creator   |  {creator}            |
| bet_place     | bet_uid       |  {bet_uid}            |
| message       | module        |  bet                  |
| message       | action        |  bet_place            |
| message       | sender        |  {creator}            |
