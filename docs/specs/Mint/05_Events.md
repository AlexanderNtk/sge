# **Events**

The **Mint Module** emits the following events:

## **BeginBlocker**

| Type  |   Attribute Key   | Attribute Value           |
|:----: |:----------------: |:-------------------------:|
| mint  | inflation         |{current_inflation}        |
| mint  | phase_provisions  |{current_phase_provision}  |
| mint  | amount            |{minted_amount}            |
